proxy(target, path, proxyFunc)

Proxy function for another function or value.
Parameters:
Name Type Description
target object The target object. properties will be copied to this object.
path string A path for the value on target.
proxyFunc function If provided, this function will be called with the returned value, and the result of this function will be returned instead.
Returns:
- The target object ( the first parameter ).
Type
any
Example
var target = { 
    value: 5,
    func(a, b){ return a + b; },
    nested: { value: 8 }
};
var proxyA = core.proxy(target, 'value');
proxyA(); // 5
var proxyB = core.proxy(target, 'func');
proxyB(); // 'ok'
var proxyC = core.proxy(target, ['nested', 'value']);
proxyC(); // 8
